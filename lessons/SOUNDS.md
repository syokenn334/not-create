# 音の名前 早見表(どれが何の音か)

レッスンに出てくる `s("bd")` などの名前が「何の音か」をまとめた表です。困ったらここを見てください。

実際に 1 つずつ鳴らして耳で確かめたいときは、同じフォルダの **[`sound-explorer.mjs`](sound-explorer.mjs)** を開き、聴きたい行の先頭の `// ` を 1 つだけ外して保存してください(選んだ音だけ鳴ります)。

## ドラム / パーカッション(`s("名前")`)

| 名前 | 音 | メモ |
|---|---|---|
| `bd` | キック(バスドラム) | 4 つ打ちの土台。低い「ドッ」 |
| `sd` | スネア | 2・4 拍の「タン」 |
| `cp` | クラップ(手拍子) | スネア代わりに 2・4 拍へ |
| `rim` | リムショット | 軽い「コッ」。パーカッションに |
| `hh` | クローズドハイハット | 刻みの「チッ」 |
| `oh` | オープンハイハット | 開いた「チー」 |
| `lt` / `mt` / `ht` | タム(低 / 中 / 高) | フィルなどに |
| `rd` | ライドシンバル | 「チーン」と鳴り続ける系 |
| `cr` | クラッシュシンバル | 展開の頭の「シャーン」 |
| `perc` | パーカッション各種 | |

同じ名前でも複数のサンプルがあります。`s("bd:3")` のように `:数字` で別のバリエーションに切り替えられます。

## シンセ波形(`note("c3").s("波形")`)

| 波形 | 音の感じ | 向いている用途 |
|---|---|---|
| `sine` | 柔らかく丸い(倍音なし) | サブベース、やさしいパッド |
| `sawtooth` | 明るく豊か(倍音多い) | ベース、リード、コードスタブ |
| `square` | 中が空っぽな「ピー」 | レトロなリード、ベース |
| `triangle` | サインよりやや明るい | やさしいメロディ |

波形の音は、M5 で学ぶフィルター(`lpf`)で倍音を削って調整します。

## もっと探す

- strudel.cc 画面のメニューにある「**sounds**」タブに、使える音の一覧が出ます。クリックで試聴・挿入できます。
- ドラム以外の音バンク例: `casio` `jazz` `metal` `east` `crow` `space` `numbers` `insect` `wind` など。`s("jazz:2")` のように使います。
- 公式の一覧と解説: strudel.cc の Workshop「First Sounds」/ ドキュメントの「Samples」。

## このカリキュラムでの使いどころ

- Phase 1(リズム編): `bd` `hh` `sd` `cp` `oh` `rim` を重ねてビートを作る。
- Phase 2 以降: `note(...).s("sawtooth")` などのシンセ波形でベース・コード・パッドを作る。
