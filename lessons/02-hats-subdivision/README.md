# M2 ハットと細分化

**今回の Strudel:** `stack` / `[]` / `~`(休符)/ `,` / `<>`
**今回の音楽理論:** サブディビジョン(8/16 分)・レイヤー・拍の位置

## レイヤーを重ねる

`stack(...)` を使うと、複数のパターンを同時に鳴らせます。

```js
setcpm(130 / 4)
stack(
  s("bd*4"),          // キック(4 分)
  s("hh*8"),          // ハイハット(8 分)
  s("~ cp ~ cp"),     // クラップを 2 拍目・4 拍目に
)
```

`"a, b"` のようにパターン文字列をカンマで区切っても同じことができます。`stack` は独立したレイヤーを明示的に書きたいときに便利です。

## サブディビジョン

1 拍を分割することをサブディビジョンといいます。

- 8 分音符 = 1 サイクルに 8 音 = `"hh*8"`
- 16 分音符 = 1 サイクルに 16 音 = `"hh*16"`

細かくするほどビートが忙しくなります。まず `"hh*8"` で 8 分ハットに慣れてから、`"hh*16"` に変えて違いを聴き比べてみましょう。

## 拍の位置と休符

`~` は休符です。`"~ cp ~ cp"` は 1 サイクルを 4 等分して、2 番目と 4 番目に cp(クラップ)を置くという意味になります。

スネアやクラップを 2 拍目・4 拍目に置くのをバックビートといい、ハウスやテクノでは定番のパターンです。位置をずらすだけでノリが大きく変わります。

## グループ `[]`

`"bd [hh hh]"` のように `[]` を使うと、一部の音符だけをさらに細分化できます。同じサイクル長のまま特定の箇所だけ密度を上げたいときに役立ちます。
