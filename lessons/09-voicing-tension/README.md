# M9 テンションとボイシング

**今回の Strudel:** `"<Cm7 ...>".voicing()` / `.rootNotes(2)` / コード記号(m7, ^7, 7)
**今回の音楽理論:** 7th/9th(テンション)/ ボイシング / ベースとの連動

## コード記号

コード記号はコードの名前と種類を短く表したものです。主な記号を覚えておくと、進行を素早く書けます。

- `Cm7` : C マイナー 7th(Cマイナー三和音 + 短 7 度)
- `Ab^7` : A♭ メジャー 7th(A♭メジャー三和音 + 長 7 度)
- `Bb7` : B♭ ドミナント 7th(B♭メジャー三和音 + 短 7 度)

三和音に 7 度の音を足すと響きが豊かになり、ジャズやハウスのスタブらしいサウンドになります。

## ボイシング

同じコードでも、構成音をどの高さでどう並べるかによって印象が大きく変わります。これをボイシングといいます。`.voicing()` は、コード記号から隣のコードへ滑らかにつながる配置を自動で選んでくれるので、手で音を並べる手間が省けます。

```js
setcpm(120 / 4)
stack(
  "<Cm7 Ab^7 Bb7 Cm7>".voicing().s("sawtooth").lpf(1500).attack(0.02).release(0.3).gain(0.4),
  "<Cm7 Ab^7 Bb7 Cm7>".rootNotes(2).s("sawtooth").lpf(500).decay(0.2).sustain(0).gain(0.6),
)
```

## テンション(9th 等)

`Cm9` のように 9th を付けると、7th に加えてさらに上の音が加わり、広がりのある響きになります。テンションはアクセントとして効果的ですが、入れすぎると響きが濁ることもあります。

## ベースとの連動

`.rootNotes(2)` は、同じコード進行のルート音をオクターブ 2 の低い位置で鳴らします。コードとベースが同じ進行を共有するため、音楽的な一体感が生まれます。引数の数字でオクターブを変えられます。
