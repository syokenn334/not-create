# M7 スケールと音程

**今回の Strudel:** `n("0 1 2 ...")` / `.scale("c4:minor")` / モード名
**今回の音楽理論:** メジャー/マイナー / 度数 / モード(ドリアン等)

## スケールと度数

スケールとは、曲の中で使う音の集合です。`n` に入れる数字は「度数」で、`scale` を指定すると 0 がルート音、1 が次の音…と自動的に並びます。半音の位置を覚えなくても、度数を使うだけで外れた音が出ません。

```js
setcpm(120 / 4)
n("0 1 2 3 4 5 6 7").scale("c4:minor").s("triangle").gain(0.5)
```

## メジャーとマイナー

同じルート音(例: C)でも、スケールによって印象が大きく変わります。メジャースケールは明るく開放的、マイナースケールは暗く緊張感があります。この違いは、3 番目の音(度数 2)が半音分高いか低いかで生まれます。

`scale("c4:major")` と `scale("c4:minor")` で同じフレーズを弾き比べると、印象の違いを直感的に感じ取れます。

## モード

マイナーやメジャー以外にも、音の並び方の種類(モード)があります。ドリアン、フリジアンなどがその代表です。ハウスやテクノのメロディには、マイナー系やドリアンが多く使われています。

```js
n("0 1 2 3 4 5 6 7").scale("c4:dorian").s("triangle").gain(0.5)
```

モードを変えるだけで、同じ度数のフレーズが違う雰囲気になります。
