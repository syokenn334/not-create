# M0 はじめの一歩

**今回の Strudel:** サイクルの考え方 / ミニ記法の読み方 / 評価して音を出す
**今回の音楽理論:** 拍と小節(時間の捉え方)

---

## サイクルとは

Strudel は時間を「サイクル」という単位で繰り返します。基本的に 1 サイクル = 1 小節と考えてください。

```js
s("bd")
```

このコードは「1 サイクルに 1 回キック(`bd`)を鳴らす」という意味です。Strudel はこれをずっとループし続けます。音楽でいえば、4 拍子の 1 小節をひたすら繰り返している状態です。

---

## ミニ記法の読み方

ダブルクォート `" "` の中に書くのが**ミニ記法**です。スペースで区切ると、1 サイクルがその数だけ等分されます。

- `"bd"` → 1 サイクルに 1 音
- `"bd hh"` → 1 サイクルを 2 等分(キック・ハット)
- `"a b c d"` → 1 サイクルを 4 等分

`example.mjs` の `s("bd")` を読み解くと:

- `s(...)` は「このサウンド名のサンプルを鳴らす」という命令
- `"bd"` は bass drum(バスドラム = キック)のサンプル名
- スペースがないので 1 サイクルにちょうど 1 回だけ鳴る

---

## やってみる

練習環境を起動して、実際に音を出してみましょう。

```powershell
$env:PATTERNS_DIR="lessons"; npm start
```

起動したら strudel.cc をブラウザで開き、注入スクリプトを実行して一度 Play します。
手順の詳細は [`../README.md` の「練習環境の起動」](../README.md#練習環境の起動) を参照してください。

`example.mjs` を保存するとブラウザにパターンが即時反映されます。キックの音が一定間隔で繰り返されることを耳で確認してください。保存すると、画面に**パンチカード**(音が鳴るタイミングを格子で表した図)が表示されます。リズムを目で確認しながら進められます。
