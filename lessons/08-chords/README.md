# M8 和音の基礎

**今回の Strudel:** `note("[c3,eb3,g3]")`(カンマ=同時)/ `<>`(コード切替)
**今回の音楽理論:** 三和音(メジャー/マイナー)/ コード進行 / `<>` の意味

## 和音とは

複数の音を同時に鳴らすと和音になります。Strudel のミニ記法では、カンマ `,` を使うと同時に鳴らせます。`"c3 eb3 g3"` は順番に鳴るのに対し、`"[c3,eb3,g3]"` は 3 つの音が同時に鳴ります。

```js
note("[c3,eb3,g3]")   // カンマ区切り = 同時に鳴る(和音)
note("c3 eb3 g3")     // スペース区切り = 順番に鳴る(メロディ)
```

## 三和音

三和音はルート、3 度、5 度の 3 つの音で構成されます。

- **メジャー(明るい):** ルート + 4 半音 + 7 半音。例: C メジャー = `[c3,e3,g3]`
- **マイナー(暗い):** ルート + 3 半音 + 7 半音。例: C マイナー = `[c3,eb3,g3]`

この例のコード進行は Cm → A♭ → B♭ → Cm で、C マイナースケール上の i → VI → VII → i です。

## コード進行と `<>`

`<>` は中身をサイクルごとに切り替える記法です。`"<コードA コードB ...>"` と書くと、1 サイクルずつコードが変わるコード進行になります。

```js
setcpm(120 / 4)
note("<[c3,eb3,g3] [ab2,c3,eb3] [bb2,d3,f3] [c3,eb3,g3]>")
  .s("sawtooth").lpf(1200).attack(0.02).release(0.4).gain(0.4)
```
