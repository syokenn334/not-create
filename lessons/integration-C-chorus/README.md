# 統合C: 1 コーラス完成

**まとめる Strudel:** `stack` / `s` `note` `n` / `.scale` / `.voicing` `.rootNotes` / ADSR・`.lpf`
**まとめる音楽理論:** リズム + ベース + コード進行 + ボイシング + テクスチャ

## 狙い

統合B で作ったグルーヴに、コード進行(スタブとパッド)と、それに合うベースを重ねます。これまでの Phase 1/2 のリズム・音色技術と、Phase 3 の和声技術を組み合わせ、まとまりのある 1 コーラスを完成させます。

## ポイント

- **コードとベースの連動:** `.rootNotes()` で同じコード進行のルート音をベースに使うと、コードとベースが一致して音楽的な根拠が生まれます。
- **スタブとパッドの役割分担:** スタブ(短い attack/decay)はリズムを刻む役割、パッド(長い release)は空間を埋めて持続感を出す役割です。2 つを組み合わせると音に奥行きが生まれます。
- **音域の整理:** `.lpf` でスタブは中域、ベースは低域に帯域を絞ることで、音が重なっても濁りません。

## 進め方

1. `brief.mjs` を開き、TODO を上から順に埋めていきます。
2. まず統合B のコードをそのまま貼り、動くことを確認します。
3. コードスタブ → パッド → ベース変更 の順で追加するとミスが見つけやすいです。
4. `rubric.md` の全項目が満たせたら完成です。
5. 完成したらギャラリーに作品として残しましょう。
