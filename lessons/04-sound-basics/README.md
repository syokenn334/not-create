# M4 サウンド基礎

**今回の Strudel:** `note` / `.s("sawtooth")`(波形)/ `.attack/.decay/.sustain/.release`
**今回の音楽理論:** 音色 / ADSR エンベロープ(音の時間的な形)

## 音を選ぶ vs 音を作る

これまで使ってきた `s("bd")` や `s("hh")` は、あらかじめ録音されたサンプル音源を鳴らす方法です。一方、`note(...).s("sawtooth")` は波形からシンセサイザーで音をゼロから作ります。

波形の種類で音色が大きく変わります:

- `sine` — 倍音を含まない、純粋で柔らかいサイン波
- `sawtooth` — 多くの倍音を持つ、明るく鋭いのこぎり波
- `square` — 奇数倍音のみを含む、中空で力強い矩形波
- `triangle` — sine に近い、柔らかくまるい三角波

## ADSR とは

ADSR は音の音量が時間とともにどう変化するかを表すエンベロープです。

```
音量
 |  /\
 | /  \___
 |/        \
 +--------- 時間
   A  D  S  R
```

- **Attack(立ち上がり)**: 音が鳴り始めてから最大音量に達するまでの時間(秒)。短いと歯切れよく、長いと「ふわっと」入る。
- **Decay(減衰)**: 最大音量からサステインレベルまで落ちる時間。
- **Sustain(保持)**: キーを押し続けている間の音量レベル(0〜1)。`0` にすると Decay 後に無音になる。
- **Release(余韻)**: 音を止めてから完全に消えるまでの時間。

**よくある組み合わせ:**

- プラック音(弦を弾いた感じ): `attack` 短い + `sustain(0)` + `decay` で調整
- パッド音(ふわっとした持続音): `attack` 長め + `sustain` 高め + `release` 長め

## example を読み解く

```js
// M4 サウンド基礎(シンセと ADSR)
setcpm(130 / 4)
note("c3 eb3 g3 c4")           // C マイナーの分散和音
  .s("sawtooth")               // のこぎり波シンセ
  .attack(0.01)                // 立ち上がり(秒)
  .decay(0.15)                 // 減衰
  .sustain(0.4)                // 保持レベル(0〜1)
  .release(0.1)                // 余韻
```

`note("c3 eb3 g3 c4")` は C マイナーの構成音を分散して鳴らします。`s("sawtooth")` で波形を指定し、ADSR で「歯切れよく短めに切れる」音色に仕上げています。
