# M1 サイクルとキック

**今回の Strudel:** `setcpm` / `*`(繰り返し)/ `s("bd*4")`
**今回の音楽理論:** 4 つ打ち・テンポ(BPM)・拍子

---

## 4 つ打ちとは

4 つ打ちはハウスやテクノの土台です。1 小節 4 拍すべてにキックを置くパターンで、安定した推進力を生みます。クラブミュージックがフロアを動かせるのは、この均等なキックのグルーヴが体に伝わるからです。

---

## テンポの指定

Strudel は「1 分間のサイクル数(cpm = cycles per minute)」でテンポを管理します。1 サイクルを 1 小節(4 拍)とするので、BPM を 4 で割った値を `setcpm` に渡します。

- ハウスの典型的なテンポ: 120–130 BPM
- 130 BPM の場合 → `setcpm(130 / 4)`

式のまま書いても計算済みの数値を書いても動きますが、`130 / 4` のように BPM を明示しておくと後から読みやすくなります。

---

## `*` の意味

`"bd*4"` の `*` は「1 サイクルに何回繰り返すか」を指定します。

- `"bd*1"` → 1 サイクルに 1 回(M0 の `"bd"` と同じ)
- `"bd*4"` → 1 サイクルに 4 回 = 4 つ打ち
- `"bd*8"` → 1 サイクルに 8 回 = 8 分音符刻み

`example.mjs` を読み解くと:

```js
// M1 サイクルとキック(4 つ打ち)
// setcpm で BPM を指定(1 サイクル = 1 小節 = 4 拍なので BPM/4)。
// "bd*4" は 1 サイクルにキックを 4 回 = 4 つ打ち。
setcpm(130 / 4)
s("bd*4")
```

- `setcpm(130 / 4)` でテンポを 130 BPM に設定
- `s("bd*4")` で 1 サイクルあたり 4 回キックを鳴らす
