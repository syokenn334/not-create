# M6 ベースライン

**今回の Strudel:** `scale("c2:minor")` / `n("0 3 5")`(度数)/ オクターブ
**今回の音楽理論:** スケール(マイナー) / 度数 / ルート / キックとベースの住み分け

## スケールと度数

スケール(音階)とは、曲で使う音の集合です。C マイナースケールは C・D・Eb・F・G・Ab・Bb という 7 つの音から構成されます。

Strudel では `scale("c2:minor")` を指定すると、`n(...)` の数字が「スケール上の何番目か」(度数)を表すようになります:

| 度数 | C マイナーでの音 |
|------|----------------|
| `0`  | C2(ルート)      |
| `1`  | D2             |
| `2`  | Eb2            |
| `3`  | F2             |
| `4`  | G2             |
| `5`  | Ab2            |
| `6`  | Bb2            |
| `7`  | C3(1 オクターブ上のルート) |

半音の位置を覚えなくても、度数を指定するだけでスケールから外れた音(外れた音 = 不協和な音)を避けられます。

## ルートとオクターブ

`0` がルート音(ここでは C2)です。`7` を指定すると 1 オクターブ上のルート(C3)になります。

ベースは低音域が担当なので、`c2`(2 オクターブ目の C)や `c1` など、低めのオクターブを使うのが一般的です。`c4` など高いオクターブを指定するとベースらしさが薄れます。

## キックと住み分け

キック(`bd`)もベースも低音域を占めるため、同時に鳴らすと音が重なって濁りやすくなります。これを「住み分け」や「棲み分け」と呼びます:

- **タイミングで住み分ける**: ベースをキックの隙間(`~`)に置く
- **長さで住み分ける**: ベース音を短く切る(`sustain(0)` でプラック化)

両方を組み合わせると、キックとベースがお互いのスペースを邪魔せず、グルーヴがハッキリします。

## example を読み解く

```js
// M6 ベースライン(スケールとルート)
setcpm(130 / 4)
stack(
  s("bd*4"),                                       // キック
  n("0 ~ 0 ~ 3 ~ 5 ~").scale("c2:minor")           // C マイナーの度数でベース
    .s("sawtooth").lpf(600).decay(0.2).sustain(0)  // 短いプラックベース
)
```

`n("0 ~ 0 ~ 3 ~ 5 ~")` はキックの合間(偶数ステップ)でベースが鳴るパターンです。`sustain(0)` で短く切り(プラック化)、フィルター(`lpf(600)`)で倍音を抑えた締まったベース音に仕上げています。
