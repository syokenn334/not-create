# M11 曲の構成

**今回の Strudel:** `arrange([サイクル数, パターン], ...)` / `cat` / `<>`
**今回の音楽理論:** 曲の構成(イントロ / ビルド / ドロップ / ブレイク)

## ループから曲へ

1 つのループを延々鳴らすだけでは曲にはなりません。時間軸でセクションを並べると「展開のある曲」になります。ハウス/テクノでは 8〜16 サイクル単位でセクションが変わるのが一般的です。

## arrange

`arrange([4, パターンA], [8, パターンB])` は A を 4 サイクル、B を 8 サイクル順に流します。セクションごとにレイヤー数を変えることで緊張と解放を作れます。

```js
// M11 曲の構成(arrange でセクションを並べる)
// [サイクル数, パターン] を順に並べる。イントロ→ビルド→ドロップ。
setcpm(124 / 4)
arrange(
  [4, s("bd*4")],                                   // イントロ: キックのみ
  [4, stack(s("bd*4"), s("hh*8"))],                 // ビルド: ハット追加
  [8, stack(s("bd*4"), s("hh*8"), s("~ cp ~ cp"))], // ドロップ: フル
)
```

このexample を読み解くと、イントロ(キックのみ 4 サイクル)→ビルド(ハット追加 4 サイクル)→ドロップ(フル 8 サイクル)という構成になっています。

## 典型的な流れ

- **イントロ:** 要素が少ない状態でスタート
- **ビルド:** 少しずつレイヤーを足して緊張を高める
- **ドロップ:** フルの状態で解放感を出す
- **ブレイク:** キックを抜いて「溜め」を作る

8 / 16 サイクル単位で区切るのが定番です。4 サイクル = 約 8 秒(124 BPM 時)。
