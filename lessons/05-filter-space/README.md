# M5 フィルターと空間

**今回の Strudel:** `.lpf`(=cutoff)/ `sine.range(min,max).slow(n)` / `.room` / `.delay`
**今回の音楽理論:** 周波数と倍音 / フィルター / 残響・ディレイ(空間)

## フィルター

音には「倍音」と呼ばれる複数の周波数成分が含まれています。ローパスフィルター(`.lpf`)は指定した周波数(カットオフ)より上の倍音を削ることで、音色を変えます。

- カットオフを下げると → 倍音が減り、暗く・こもった音になる
- カットオフを上げると → 倍音が増え、明るく鋭い音になる

```js
.lpf(800)        // 800Hz 以上をカット
.lpf(200)        // さらに暗い、こもった音
.lpf(4000)       // ほぼ素の音(倍音が通る)
```

`.lpf` の第 2 引数で **resonance**(レゾナンス)も指定できます。カットオフ付近の周波数を強調し、「ピキッ」とした独特の鳴りが加わります(例: `.lpf(800, 0.5)`)。

## フィルターを動かす

フィルターを固定値ではなく、時間とともに変化する信号で動かすと「展開」が生まれます。`sine.range(300, 1500).slow(4)` は:

- `sine` — 0〜1 のサイン波信号
- `.range(300, 1500)` — その信号を 300〜1500 の範囲に変換
- `.slow(4)` — 4 サイクル(= 4 小節)かけてゆっくり 1 往復

この LFO(低周波振動)によってフィルターが開閉し、単調なループに自然な起伏が生まれます。

## 空間

`.room(0.3)` は残響(リバーブ)で、音に空間的な広がりを加えます。値は 0(ドライ)から 1(非常に広い空間)。かけすぎると音が濁り輪郭が失われるので、0.2〜0.4 程度から始めるのがおすすめです。

`.delay(0.25)` はやまびこ(ディレイ)で、音を少し遅らせて繰り返します。値はディレイのウェット量。こちらもかけすぎ注意です。

## example を読み解く

```js
// M5 フィルターと空間
setcpm(130 / 4)
note("c2*8").s("sawtooth")
  .lpf(sine.range(300, 1500).slow(4))  // カットオフを LFO でゆっくり上下
  .room(0.3)                           // 残響(空間)
  .delay(0.25)                         // ディレイ(やまびこ)
```

`c2*8` は C2 の音を 1 サイクルに 8 回鳴らす、のこぎり波のベース音です。フィルターが 4 小節かけてゆっくり開閉し、残響とディレイで音に奥行きが加わります。
