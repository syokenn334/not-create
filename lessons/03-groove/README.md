# M3 グルーヴ

**今回の Strudel:** `.gain` / `.swing` ・ `.swingBy` / `.euclid` / `.room`
**今回の音楽理論:** スウィング・シンコペーション・アクセント・ユークリッドリズム

## サンプルコード

```js
// M3 グルーヴ(スウィング・アクセント・ユークリッド)
setcpm(130 / 4)
stack(
  s("bd*4"),
  s("hh*8").gain("0.5 0.8").swing(4),   // 8 分ハット、強弱(アクセント)+スウィング
  s("~ cp ~ cp").room(0.25),            // クラップ(2・4 拍)に少し残響
  s("rim").euclid(3, 8).gain(0.5),      // ユークリッド(3/8)のパーカッション
)
```

## アクセント(強弱)

`.gain("0.5 0.8")` のように値をパターンにすると、音ごとに音量が変わりノリが出ます。8 分ハットなら `"0.5 0.8"` を繰り返すことで 1 音ごとに弱・強を交互に付けられます。等音量で並ぶだけのハットより人間らしいグルーヴになります。

## スウィング

音符を等間隔に並べると機械的な印象になりがちです。`.swing(4)` は `.swingBy(1/3, 4)` と等価で、4 分割の後ろの音を少し遅らせることではね感(スウィング感)を加えます。引数の数字はサイクルの分割数なので、8 分ハットに掛けるなら `.swing(8)` も試してみてください。

## ユークリッドリズム

`.euclid(3, 8)` は「8 枠に 3 音をなるべく均等に配置する」という意味です。手で書くと `[x . . x . . x .]` のようなパターンになり、西アフリカやラテン音楽でよく見られる反復的なリズムを簡単に作れます。最初の数字(音の数)と 2 番目(全体の枠数)を変えるだけでパーカッションの密度が変わります。

## 残響

`.room(0.25)` でリバーブ(残響)を加えます。クラップのような鋭い音に少し空間感を足すと自然になります。ただしやりすぎると音がにごるので小さめの値から始めましょう。
