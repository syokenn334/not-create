## 展開の付け方

単純な 1 小節ループは飽きる。2 小節のうねりを作る方法が 2 つある。

`<>` を使うと、サイクルごとに交替するパターンが書ける。

```js
s("<bd*4 [bd*4 bd*2]>")
// 1 サイクル目: bd*4 / 2 サイクル目: [bd*4 bd*2]
```

`every` を使うと、N サイクルに 1 回だけ変化を加えられる。

```js
s("hh*8").every(2, x => x.fast(2))
// 2 サイクルに 1 回、ハイハットが倍速になる
```

どちらか、または組み合わせて使うと良い。
