# 統合B: グルーヴ完成

**まとめる Strudel:** `stack` / `s` `note` `n` / `.scale` / `.s` `.adsr`系 / `.lpf`+`sine.range` / `.room` `.delay`
**まとめる音楽理論:** リズム + ベース + 音色 + フィルター展開

## 狙い

統合A で作ったドラムパターンに、スケールに沿ったベースラインと音作り・フィルター展開を重ね、低域がぶつからない 1 つのグルーヴを完成させます。Phase 1 で学んだリズムと、Phase 2 で学んだ音色・ベースを統合する課題です。

## ポイント

- **キックとベースの住み分け(M6)**: ベースをキックの隙間に置く / `sustain(0)` で短く切る。低域が濁らないことがグルーヴの土台になります。
- **フィルターの動きで単調さを避ける(M5)**: `lpf(sine.range(...).slow(...))` をどこかのレイヤーに仕込むと、ループが長く聴けるようになります。
- **ベース音色の ADSR(M4)**: `sawtooth` + 短い `decay` + `sustain(0)` でプラックベースが作れます。波形と ADSR を変えてみましょう。

## 進め方

`brief.mjs` の TODO を上から順に埋めていきます:

1. 統合A のドラム(ハット / クラップ / パーカッション)を `stack` の中に貼りつける
2. `n(...).scale("c2:minor")` でベースラインを追加し、キックとぶつからないよう調整する
3. ベースに `.s("sawtooth")` + ADSR を設定して音色を整える
4. どこかのレイヤー(ベースか別のレイヤー)に `lpf(sine.range(...).slow(...))` を加えて展開を付ける

`rubric.md` の全項目を満たしたら完成です。気に入ったらギャラリーに作品として残しましょう。
